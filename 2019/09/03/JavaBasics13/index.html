String

String

字符串，用一对””表示

  • String类用final修饰，无法被继承
  • String实现了Serializable序列化接口，支持序列化和反序列化操作（IO）
  • String实现了Comparable接口，String是支持比较大小的
  • String类中定义了private final char value[]；存放字符串的值(String底层通过字符数组来存放字符序列)

String是不可变的字符序列，String具有不可变性

String的创建方式：

1.字面量赋值
如果String采用的字面量方式进行赋值，那么该字符串存储于方法区的常量池中。而常量池中，数据一般是不重复的
2.String(String original) 把字符串数据封装成字符串对象 this.value = original

class Test{
public static void main(String[] args){ 
String s1 = "Greed";
String s2 = "Greed";
System.out.println(s1==s2);//true 字面量方式赋值，地址相同
s2 = "Ring";
System.out.println(s1==s2);//false String重新赋值时，在常量池中新开辟一块区域，原区域不变
String s3 = new String("Greed");
String s4 = new String("Greed");
System.out.println(s1==s3);//false
/*通过构造方法赋值时，会在堆区创建一个value属性
value属性指向常量池中的Greed的地址，s3指向value属性的地址，s1指向常量池中的Greed的地址*/
System.out.println(s3==s4);//false 构造方法赋值时，每次都会在堆区创建一个新的value属性，s3和s4指向不同的value属性
  }
}
  • String(char[] value) 把字符数组的数据封装成字符串对象 this.value = Arrays.copyOf(value);
  • String(char[] value,int index,int count) 把字符数组中的一部分数据封装成字符串对象 this.value = Arrays.copyOf(value,start,length);
class Test{
public static void main(String[] args){
  char[] arr = {'i','l','o','v','e'};
String s = new String(arr);
System.out.println(s);//ilove
String s_ = new String(arr,1,4);//从下标为1开始，长度为4
System.out.println(s_);//love
  }
}

字符串拼接

字符串拼接过程中，一旦出现了变量，那么最终拼接之后的字符串保存在堆中

class Test{
    public static void main(String[] args) {
        String s1 = "hello";
        String s2 = "world";
        String s3 = "helloworld";

        String s4 = "hello"+"world";
        String s5 = s1+"world";
        String s6 = "hello"+s2;
        String s7 = s1+s2;
        System.out.println(s3==s4);//true s4为常量拼接
        System.out.println(s3==s5);//false s5为变量s1与常量world拼接，保存在堆中
        System.out.println(s3==s6);//false 理由同上
        System.out.println(s3==s7);//false 理由同上
  }
}

  • 截取到指定位置结束（前闭后开，包含beginIndex，不包含endIndex） String substring(int beginIndex，int endIndex)

String类补充方法

1.比较两个字符串的字典
int compareTo(String anotherString)
首先，先比较两个字符串第一个字母，如果第一个字母不一样，返回字母对应的ascii码值的差值
如果当前位置上字母相同，则比较后一位,直到末位
如果字符串长度不同，前同位字母也相同，则返回俩字符串长度差值
如果每一位都相同且长度相等，返回0

String s1 = "abc";//97 98 99
String s2 = "adfe";//97 100 102 101
String s3 = "ab";
System.out.println(s1.compareTo(s2));//-2
System.out.println(s1.compareTo(s3));//1

2.将指定的字符串拼接在字符串的末尾 String concat(String str)

如何反转一个字符串

public class Test {
    public static void main(String[] args) {
        String str = "greedring";
        System.out.println(flip(str));
    }

    //该方法比连续的字符串拼接省内存
    private static String flip(String str) {
        char[] arr = str.toCharArray();//将字符串转化为数组
        for (int start = 0, end = arr.length - 1; start < end; start++, end--) {
            char temp = arr[start];
            arr[start] = arr[end];
            arr[end] = temp;
        }
        return new String(arr);
    }
}

StringBuilder

String：不可变字符序列
StringBuffer和StringBuilder：可变字符序列
两者功能基本相似，StringBuffer线程安全，StringBuilder线程不安全（效率高）

总的效率来说：StringBuilder > StringBuffer > String

StringBuilder是一个可变字符串，字符串缓冲区。
StringBuilder底层数组可以动态改变数组中元素，还能改变数组自身引用。
String的内容固定，而StringBuilder的内容可变。

StringBuilder常用方法

1.构造方法
创建StringBuilder对象：StringBuilder 对象名 = new StringBuilder();

2.成员方法

  • 添加数据，并返回自身对象 public StringBuilder append(任意类型)
    如果进行append操作时，底层字符数组如果容量不够存放，会进行自动扩容操作，
    默认会扩容到原有长度的两倍+2长度

  • 返回长度(已经存储的字符个数) public int length()
    底层其实返回的count值（记录了数组中实际使用了多少长度，未使用的为null）

  • 返回当前容量 (理论值) 数组的实际大小 public int capacity()
    底层其实返回的是字符数组的长度（不管有没有被使用）
    在创建新的StringBuilder数组时，默认创建长度为16。

  • 反转功能 public StringBuilder reverse()
    反转自身，不会创建新对象

public class Test {
    public static void main(String[] args) {
        //1.默认创建一个长度为16的数组
        StringBuilder sb = new StringBuilder();
        StringBuilder sb1 = sb.append("hello");
        System.out.println(sb == sb1);//true

        System.out.println("数组实际容量:" + sb.capacity());//16
        System.out.println("数组实际常用长度：" + sb.length());//0
        sb.append("hello");
        System.out.println("数组实际容量:" + sb.capacity());//16
        System.out.println("数组实际常用长度：" + sb.length());//5
        sb.append("world");
        //允许链式调用
        sb.append("greedring").append(true).append(7).append('q');//自动扩容
        System.out.println("数组实际容量:" + sb.capacity());//34
        System.out.println("数组实际常用长度：" + sb.length());//30
        System.out.println(sb);//StringBuilder重写了toString方法，等同于sb.toString()

        //2.底层默认字符串长度+16的长度的字符数组
        StringBuilder sb2 = new StringBuilder("helloworld");
        System.out.println("数组实际容量:" + sb2.capacity());//10+16=26
        System.out.println("数组实际常用长度：" + sb2.length());//10
        sb2.append("dsajkdnasdsanasdsad");
        System.out.println("数组实际容量:" + sb2.capacity());//26*2+2=54
        System.out.println("数组实际常用长度：" + sb2.length());//29

        //3.参数：底层数组的实际容量（自定义底层数组长度）
        StringBuilder sb3 = new StringBuilder(20);
        System.out.println("数组实际容量:" + sb3.capacity());//20
        System.out.println("数组实际常用长度：" + sb3.length());//0

        StringBuilder sb4 = new StringBuilder("greedring");
        sb4.reverse();//反转自身，不创建新对象
        System.out.println(sb4.toString());//这里的.toString()方法可省略，Stringbuilder已重写
    }
}

例题

1.利用StringBuilder把数组拼接成一个字符串

public class Test {
    public static void main(String[] args) {
        int[] arr = {1, 2, 3};
        System.out.println(arrayToString(arr));
    }

    private static String arrayToString(int[] arr) {
        StringBuilder sb = new StringBuilder(arr.length);
        for (int i = 0; i < arr.length; i++) {
            if (i == 0) {
                sb.append("[").append(arr[i]).append(",");
            } else if (i < arr.length - 1) {
                sb.append(" ").append(arr[i]).append(",");
            } else {
                sb.append(" ").append(arr[i]).append("]");
            }
        }
        return sb.toString();
    }
}

2.判断一个字符串是否是对称字符串

public class Test {
    public static void main(String[] args) {
        String str = "abcba";
        if (getContrary(str)) {
            System.out.println("是对称字符串");
        } else {
            System.out.println("不是对称字符串");
        }
    }

    private static boolean getContrary(String str) {
        StringBuilder sb = new StringBuilder(str);
        sb.reverse();
        return str.equals(sb.toString());
    }
}
