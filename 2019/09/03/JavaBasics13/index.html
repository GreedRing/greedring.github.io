2019年09月03日 701 字 大概 3 分钟 Java Java基础

String

String

字符串，用一对””表示

  • String类用final修饰，无法被继承
  • String实现了Serializable序列化接口，支持序列化和反序列化操作（IO）
  • String实现了Comparable接口，String是支持比较大小的
  • String类中定义了private final char value[]；存放字符串的值(String底层通过字符数组来存放字符序列)

String是不可变的字符序列，String具有不可变性

String的创建方式：

  • 字面量赋值
    如果String采用的字面量方式进行赋值，那么该字符串存储于方法区的常量池中。而常量池中，数据一般是不重复的
  • String(String original) 把字符串数据封装成字符串对象 this.value = original
class Test{
public static void main(String[] args){ 
String s1 = "Greed";
String s2 = "Greed";
System.out.println(s1==s2);//true 字面量方式赋值，地址相同
s2 = "Ring";
System.out.println(s1==s2);//false String重新赋值时，在常量池中新开辟一块区域，原区域不变
String s3 = new String("Greed");
String s4 = new String("Greed");
System.out.println(s1==s3);//false
/*通过构造方法赋值时，会在堆区创建一个value属性
value属性指向常量池中的Greed的地址，s3指向value属性的地址，s1指向常量池中的Greed的地址*/
System.out.println(s3==s4);//false 构造方法赋值时，每次都会在堆区创建一个新的value属性，s3和s4指向不同的value属性
  }
}
  • String(char[] value) 把字符数组的数据封装成字符串对象 this.value = Arrays.copyOf(value);
  • String(char[] value,int index,int count) 把字符数组中的一部分数据封装成字符串对象 this.value = Arrays.copyOf(value,start,length);
class Test{
public static void main(String[] args){
  char[] arr = {'i','l','o','v','e'};
String s = new String(arr);
System.out.println(s);//ilove
String s_ = new String(arr,1,4);//从下标为1开始，长度为4
System.out.println(s_);//love
  }
}

字符串拼接

字符串拼接过程中，一旦出现了变量，那么最终拼接之后的字符串保存在堆中

class Test{
    public static void main(String[] args) {
        String s1 = "hello";
        String s2 = "world";
        String s3 = "helloworld";

        String s4 = "hello"+"world";
        String s5 = s1+"world";
        String s6 = "hello"+s2;
        String s7 = s1+s2;
        System.out.println(s3==s4);//true s4为常量拼接
        System.out.println(s3==s5);//false s5为变量s1与常量world拼接，保存在堆中
        System.out.println(s3==s6);//false 理由同上
        System.out.println(s3==s7);//false 理由同上
  }
}

字符串判断

  • boolean equalsIgnoreCase(String str):比较字符串的内容是否相同,忽略大小写
    String s1 = “greed”;
    String s2 = “Greed”;
    s1.equals(s2);//true

  • boolean startsWith(String str):判断字符串对象是否以指定的str开头
    String s1 = “greed”;
    s1.startsWith(“gr”);//true

  • boolean endsWith(String str):判断字符串对象是否以指定的str结尾
    String s1 = “greed”;
    s1.endsWith(“eed”);//true
