集合引入

1.集合：集合（数组）是对多个数据进行存储操作的结构，Java容器

注意：此处的存储是基于内存结构的存储，不同于持久化存储（txt数据库….）

2.数组特点

int[] arr = new int[5];
User[] arr = new User[5];//对象数组
Object[] objects = new Object[5];

  • 数组一旦初始化，长度就固定了（ArrayIndexOutOfBoundsException）
  • 数组一旦定义，那么数组中存储的数据元素的数据类型就确定了

3.数组的缺点：

  • 数组一旦初始化，长度就固定了，无法进行扩容
  • 数组中提供的操作方法过少，删除、插入元素效率过低
  • 数组中没有提供获取数组实际长度的操作
  • 数组中数据默认是有序且可以重复的，如果需要设计无序且不能重复则无法实现

集合概述

体系结构：

  • List(子接口) 有序且可重复
     - ArrayList 基于“动态”数组，读取效率高(根据下标、索引)，增删效率慢
     - LinkedList 基于双向链表结构，增删效率高，读取效率慢
     - Vector 基于“动态”数组（线程安全效率低）
  • Set(子接口) 无序不可重复
     - HashSet
     - LinkedHashSet
     - TreeSet


 - HashMap
 - LinkedHashMap
 - TreeMap
 - HashTable
 - Properties

Collection方法

  • boolean  
    向集合A中添加元素
  • boolean  
    将集合B中的所有元素添加到集合A中
  • void  
    从集合A中移除所有的元素
  • boolean  
    如果集合A包含指定元素，返回true
  • boolean  
    如果集合A包含指定集合B的所有元素，返回true
  • boolean  
    将集合A与集合B进行比较，以进行相等性比较（顺序需要依次对应）
  • int  
    返回集合A的哈希代码值
  • boolean  
    如果集合A不包含任何元素，返回true
  • boolean  
    从集合A中移除指定元素的一个实例，如果它是存在的
  • boolean  
    删除集合A中包含的所有，集合B中的元素（可选操作）
    [当集合A中包含多个相同的元素，该元素在集合B中，全部删除]
  • boolean  
    求两个集合的交集,并将交集返回给集合A
    [如果俩集合相等，返回false]
  • int  
    返回集合A中的元素的数目
  • Object[]  
    返回包含集合A中所有元素的数组

  • <T> T[]

package com.greedring;

import java.util.ArrayList;
import java.util.Collection;
import java.util.Date;
import java.util.Iterator;

public class Test {
    public static void main(String[] args) {
        Collection collection = new ArrayList();
        collection.add(123);
        collection.add("Yae");
        collection.add('a');
        collection.add(123);
        collection.add(new Date());
        collection.add('a');
        System.out.println("创建的集合为："+collection);

        //获取迭代器对象，每次获取默认会存在一个游标，指向集合第一个元素之前的位置
        Iterator iterator = collection.iterator();
        //每次调用next()方法，会实现游标后移，并返回当前游标指向的对象
        Object object = iterator.next();
        System.out.println(object);

        //判断当前游标的下一个是否有元素,存在返回true，不存在返回false
        while (iterator.hasNext()) {
            System.out.println(iterator.next());
        }
        System.out.println("===================================");

        //在遍历时删除元素
        Iterator iterator1 = collection.iterator();
        while (iterator1.hasNext()) {
            Object object1 = iterator1.next();
            if (object1 instanceof Integer && (Integer) object1 == 123) {
                iterator1.remove();//删除当前游标指向的元素
            }
        }
        System.out.println("遍历集合删除123后为："+collection);

        collection.remove('a');//仅删除集合出现的第一个'a'元素
        System.out.println("删除集合中第一次出现的元素'a'后为："+collection);
    }
}

底层分析

ArrayList
 懒加载:节约内存空间
 ArrayList list = new ArrayList(); 底层创建一个空的数组（长度为0）
 list.add(123); 如果是第一次add，那么在add过程中会自动进行扩容，长度为10
 list.add(“abc”); 再次add时，会判断当前数组容量是否足够存放，如果不够，也会默认扩容到原有数组长度的1.5倍，并且会将数组内容拷贝到新的数组中

LinkedList
 基于双向链表结构
 LinkedList linkedList = new LinkedList();
 LinkedList对象默认维护了两个对象Node First、Node Last 默认值都是null，分别代表链表的头指针以及尾指针

 确认双向链表结构：Node
 E item;//当前节点的元素
 Node next;//后置指针（下一个节点元素）
 Node prev;//前置指针（指向）

Node(Node prev, E element, Node next) {
this.item = element;
this.next = next;
this.prev = prev;
}

Vector底层实现（线程安全）
 底层依旧是“动态”数组
 Vector vector = new Vector(); //底层自动初始化一个长度为10的数组结构
 vector.add(123); //如果底层数组容量不够，自动扩容原有数组长度2倍，并拷贝数组到新的数组中

ArrayList特有方法

1.void  
在集合A的指定位置插入元素,后续元素依次后移

2.boolean  
将指定的集合中的所有元素插入到集合A指定位置的列表中

3.E  
返回集合A中指定位置的元素

4.int  
查找元素在集合A中出现的第一个位置，如果找不到返回-1

5.int  
查找元素在集合A中出现的最后一个位置，如果找不到返回-1

6.E  
按照索引移除元素
[如果需要删除第一次出现的给定元素（如删除4这个元素）：集合A.remove(new Integer(4));]

7.E  
更新指定索引的值  返回旧值

8.List  
截取集合A的子集合  首索引到尾索引，前闭后开

package com.greedring;

import java.util.ArrayList;
import java.util.Date;
import java.util.List;

public class Test {
    public static void main(String[] args) {
        List list = new ArrayList();
        list.add(123);
        list.add(4);
        list.add("greed");
        list.add("ring");
        list.add(new Date());
        list.add("abc");
        System.out.println(list);
        System.out.println("===================");

        //在指定位置插入元素
        list.add(1, 'a');
        System.out.println("指定位置插入元素后：" + list);
        System.out.println("===================");

        //在指定位置插入集合中所有元素
        List list1 = new ArrayList();
        list1.add('b');
        list1.add('a');
        list.addAll(2, list1);
        System.out.println("指定位置插入集合后：" + list);
        System.out.println("===================");

        //获取集合指定位置元素
        System.out.println(list.get(0));
        System.out.println(list.get(1));
        System.out.println(list.get(8));
        System.out.println("===================");

        //查找元素在集合中出现的位置，如果找不到返回-1
        System.out.println(list.indexOf("abc"));
        System.out.println(list.lastIndexOf('a'));
        int index = list.lastIndexOf('c');
        System.out.println(index);//-1
        System.out.println("===================");

        //按照引索移除元素
        list.remove(0);
        System.out.println("移除第一个元素后：" + list);
        list.remove(new Character('a'));
        System.out.println("移除第一个a元素后：" + list);
        System.out.println("===================");

        //更新指定索引的元素
        Object set = list.set(1, 'c');
        System.out.println(set);
        System.out.println("更新元素后：" + list);
        System.out.println("===================");

        //截取子集合
        List new_List = list.subList(3, 5);
        System.out.println(new_List);
        System.out.println(list);
    }
}

LinkedList特有方法

基于双向链表结构，所以存在一些链表操作特有的方法

1.void  
在首尾插入指定元素

2.E  
返回首位元素

package com.greedring;

import java.util.LinkedList;

public class Test {
    public static void main(String[] args) {
        LinkedList linkedList = new LinkedList();
        linkedList.add(123);
        linkedList.add("Yae");
        linkedList.add('a');
        //Collection和List接口中的方法都会继承
        linkedList.add(1, 666);
        System.out.println(linkedList);

        //在列表开始处插入指定元素
        linkedList.addFirst(1);
        System.out.println(linkedList);
        //在列表开结束插入指定元素
        linkedList.addLast(999);
        System.out.println(linkedList);
        //返回列表中的第一个元素
        System.out.println("first=>" + linkedList.getFirst());
        //返回列表中的最后一个元素
        System.out.println("last=>" + linkedList.getLast());
    }
}

增强for循环

package com.greedring;

import java.util.Arrays;
import java.util.Collection;
import java.util.Date;

public class Test {
    public static void main(String[] args) {
        int[] arr = new int[]{1, 3, 6, 8, 10};
        //普通的for循环，遍历数组的长度0-长度-1，通过下标取值
        for (int i = 0; i < arr.length; i++) {
            System.out.print(arr[i] + "\t");
        }
        System.out.println();

        //增强for循环，可以不需要通过下标取值
        for (int value : arr) {
            System.out.print(value + "\t");
        }
        System.out.println();

        //集合遍历方式，底层也是采用的Iterator的方式
        Collection collection = Arrays.asList(1, "6", true, 'w', new Date());
        for (Object object : collection) {
            System.out.print(object + "\t");
        }
    }
}

可变参数

 1.格式： 数据类型… 变量名  [在形参声明中使用]
 2.可变参数传递实参时，可以传递一个数组的实参（可变参数使用类似数组 length args[i]）
 3.可变参数在调用方法时，可以传递0个、1个、2个…数量的参数
 4.可变参数的方法与同名的方法可以构成重载(但是存在匹配方法时，会优点调用匹配的非可变参数方法)
 5.可变参数的方法与同名的数组作为参数的方法不构成重载
 6.可变参数必须声明在形参列表的最后一个
 7.在一个方法的参数列表中，只能声明一个可变参数

package com.greedring;

public class Test {
    public static void main(String[] args) {
        test();
        test(1);
        test(1, 2);
        test(1, 2, 3);
    }

    private static void test() {
        System.out.println("无参函数被调用");
    }

    private static void test(int a) {
        System.out.println(a + "...test");
    }

    private static void test(int a, int b) {
        System.out.println(a + "...test..." + b);
    }

    // 用可变参数（结构类似数组）代替数组
    private static void test(int... args) {
        System.out.println("可变参数函数被调用了");
        for (int arg : args) {
            System.out.print(arg + "\t");
        }
    }
}

栈与队列

进出栈分析

先进后出

package com.greedring;

import java.util.LinkedList;

public class Test {
    //创建栈的结构
    private LinkedList data = new LinkedList();

    private LinkedList getData() {
        return data;
    }

    public void setData(LinkedList data) {
        this.data = data;
    }

    //压栈
    private void push(Object object) {
        data.add(object);
    }

    //出栈
    private Object pop() {
        return data.removeLast();
    }

    //返回栈顶元素
    private Object peek() {
        return data.getLast();
    }

    public static void main(String[] args) {
        //创建栈结构
        Test stack = new Test();
        //压栈
        stack.push(1);
        stack.push(2);
        System.out.println(stack.getData());
        Object object = stack.pop();//出栈
        System.out.println("出栈元素：" + object);
        System.out.println(stack.getData());
        System.out.println("栈顶元素：" + stack.peek());
    }
}

进出队列分析

先来后到

package com.greedring;

import java.util.LinkedList;

public class Test {
    private LinkedList data = new LinkedList();

    private LinkedList getData() {
        return data;
    }

    public void setData(LinkedList data) {
        this.data = data;
    }

    //加入队列
    private void offer(Object object) {
        data.add(object);
    }

    //离开队列
    private Object poll() {
        return data.removeFirst();
    }

    //查询出即将要离开队列的是哪个元素,检查有没有要离开队列的元素
    private Object peek() {
        return data.getFirst();
    }

    public static void main(String[] args) {
        //创建队列结构
        Test queue = new Test();

        queue.offer(1);
        queue.offer(2);
        System.out.println(queue.getData());

        Object object = queue.poll();
        System.out.println("出队列元素：" + object);
        System.out.println(queue.getData());
        System.out.println("队列顶元素：" + queue.peek());
    }
}
