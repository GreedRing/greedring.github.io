MySQL

SQL

表

class Student{
String stuName;
String stuNo;
}

stuName stuNo
Kiana 1
Mei 2

Student：表
表是类在数据库的结构表示
字段是类的属性
student(stuName,stuNo)
字段=类型+名称

类型：

  • 整数:int；

  • 小数:float；

  • 字符串：
    定长：char(固定长度)
    不定长：varchar(最大长度)

  • 日期：date

表结构创建：主键：唯一+不能为空 [所有命名不区分大小写]
creat table 表名(
字段名 字段类型,
字段名 字段类型
)

创建学生表
creat table student(
stuName VARCHAR(20),
stuNo char(10)
)

表数据
一行 = 一条记录 = java对象

表数据的操作

1.sql语句分为几类
DDL：定义语句
DCL：控制语言
DML：更新语句（u，i，d）
DQL：查询语句（s）

2.表结构的创建，修改表结构（修改列名，列类型，增加列，删除列）
alter table 表名 modify/change/add/drop 列名 新的类型/旧的列名 新的列名 列数据类型/column 列名 类名/列名；

3.查看我们自己创建的数据库
show creat database 数据库名;

4.查看正在使用的数据库
select `database`();

5.使用数据库/选择某个数据库[dos下使用]
use 数据库名;

6.查看表结构
desc 表名;
desc student;

7.删除表结构
drop table 表名;
