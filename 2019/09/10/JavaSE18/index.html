双列集合

Map

接口，双列集合（存储一对一对的数据）类似函数y=f(x)

Map中 key、value 的特点
以HashSet为例：

  • Key是无序且不可重复，如果需要获取一个Map的所有的Key，可以使用的是Set结构
  • Value是无序可重复的，如果需要获取一个Map的所有Value，可以使用Collection结构存储
  • 一个Map中的Entry对象无序且不可重复的，如果需要获取一个Map的所有键值对(Entry)，可以使用Set结构

HashMap底层实现原理：

    HashMap map = new HashMap(); //当创建HashMap时，底层默认创建一个Entry数组，并进行默认初始化，初始化长度为
    map.put(k1,v1);

  • 先调用k1对象的hashCode方式，获取k1对象的hashCode方法，获取k1的哈希值，根据某种算法得到索引值。判断当前索引位置上是否有元素，如果当前索引位置上没有元素，k1，v1直接添加成功。
  • 若前索引位置上存在一个或多个元素，将k1的哈希值依次与这一个或多个已经存在的key的哈希值进行比较，如果k1的哈希值与已存在元素的哈希值都不同，k1、v1添加成功。
  • 如果k1的哈希值与已经存在的元素中的任意一个元素的key的哈希值相同(k2,v2)，调用equals方法比较k1、k2，如果返回false，则k1、v1添加成功。
  • 如果equals方法返回true，那么v1值将覆盖v2值。
  • 对于put过程（添加元素），可能会涉及到扩容操作，当集合中键值对地方数量超过临界值，并且当前数组不为空的时候，需要进行扩容，扩容到原来的2倍，将将原有数组中数据拷贝到新的数组中。

术语（常量）：
DEFAULT_INITIAL_CAPACITY      默认初始化长度 16
DEFAULT_LOAD_FACTOR      默认加载因子 0.75（计算临界值（threshold）=16*0.75=12）
TREEIFY_THRESHOLD       JDK1.8  桶中节点数量树化最大值（链表==>红黑树）
MIN_TREEIFY_CAPACITY     64  树化数组的最小长度

JDK1.8 与 JDK 1.7区别：

(1).JDK1.7中底层是Entry[]数组 JDK1.8中底层是Node<K,V>[] table

static class Node<K,V> implements Map.Entry<K,V> {
   final int hash;
   final K key;
   V value;
   Node<K,V> next;
}

(2).JDK1.7 当调用HashMap()构造函数时，底层会创建一个长度为16的Entry数组；而JDK1.8 当调用构造函数时，并不会初始化底层数组，当第一次调用put操作时，初始化长度为16的Node[]数组。

(3).JDK1.7 桶中的数据是新增数据指向原有数据；JDK1.8 是原有数据指向新增数据。

(4).JDK1.8 在某种情况（当前数组索引下节点的长度超过了8并且数组长度超过了64）下，会将桶中（数组中某个索引位置）的数据由链表结构转换为树形结构（树化）。

Map常用方法

1.map的操作（新增、删除、修改）

  • V 
    将指定的值与集合A中的指定键关联。
  • void 
    从集合A中移除所有的映射。
  • void 
    从集合B映射到集合A，复制所有的映射；如果集合B映射到集合A中的相同键，用集合B的值覆盖集合A的。
  • V 
    如果存在，则从该集合A中移除一个键的映射。

2.map获取

  • V 
    返回集合A中指定的键映射的值；如果集合A不包含的键映射，返回null。
  • boolean 
    如果集合A包含一个指定的键映射，返回true。
  • boolean 
    如果集合A中映射到指定值的一个或多个键，返回true。
  • int 
    返回集合A中的键值映射的数目。
  • boolean 
    如果集合A不包含键值的映射，返回true。
  • boolean 
    将指定的对象与此映射的相等性进行比较。

3.map的元操作

  • Set 
    获取集合A所有的key。
  • Collection 
    获取集合A所有的value。
  • Set<Map.Entry<K,V>> 
    获取集合A所有的键值对。
package com.greedring;

import java.util.Collection;
import java.util.HashMap;
import java.util.Iterator;
import java.util.Map;
import java.util.Set;

public class Test {
    public static void main(String[] args) {
        //以HashMap为例演示Map中常用方法
        Map map = new HashMap();
        //新增元素
        map.put(1, "AA");
        map.put("bb", "BB");
        map.put("cc", "CC");
        map.put("CC", "CC");
        map.put(new User("1", "Yae"), "666");
        System.out.println(map);
        //修改
        map.put(new User("1", "Yae"), "888");
        map.put("user", new User("1", "Yae"));
        map.put(1, "AAAA");
        System.out.println(map);

/*        //清除
        map.clear();//移除集合中内容 （注意移除后并不是null）
        System.out.println(map);*/

        //新集合中所有Entry添加到原集合中
        Map map2 = new HashMap();
        map2.put("bb", "map1");
        map2.put("new", "map1");
        map.putAll(map2);
        System.out.println(map);

/*        //以某一个key移除键值对,并返回移除的value
        Object remove = map.remove(1);
        System.out.println("remove:"+remove);
        System.out.println(map);*/

        System.out.println("*************************************************");

        //获取
        //根据指定key获取value
        Object object = map.get(1);
        System.out.println(object);
        Object object2 = map.get("666");//如果不存在某个key返回null
        System.out.println(object2);
        Object object3 = map.get(new User("1", "Yae"));//匿名地址不同
        System.out.println(object3);

        System.out.println("=================================================");

        //判断key是否存在 （key使用set存储） 底层调用hashCode、equals
        boolean containsKey = map.containsKey("bb");
        System.out.println(containsKey);
        boolean containsKey2 = map.containsKey(new User("1", "Yae"));
        System.out.println(containsKey2);
        //判断value是否存在  （value使用collection存储） 底层调用equals
        boolean containsValue = map.containsValue("CC");
        System.out.println(containsValue);
        boolean containsValue2 = map.containsValue(new User("1", "Yae"));
        System.out.println(containsValue2);

        System.out.println("=================================================");

        //获取键值对的数量
        System.out.println(map.size());

        System.out.println("=================================================");

        //判断集合中是否不包含键值对
        System.out.println(map.isEmpty());//存在键值对返回false
        map.clear();
        System.out.println(map.isEmpty());

        System.out.println("=================================================");

        //判断两个集合中元素是否相同
        Map map3 = new HashMap();
        map3.put("a", "AA");
        map3.put("b", "BB");
        map3.put("c", "CC");
        Map map4 = new HashMap();
        map4.put("a", "AA");
        map4.put("b", "BB");
        map4.put("c", "CC");
        System.out.println(map3.equals(map4));
        map4.put("d", "CC");
        System.out.println(map3.equals(map4));

        System.out.println("=================================================");

        //map元操作
        //获取所有的key
        Set keySet = map4.keySet();
        System.out.println(keySet);

        //获取所有的value
        Collection values = map4.values();
        System.out.println(values);

        //获取所有的键值对
        Set entrySet = map4.entrySet();
        System.out.println(entrySet);

        System.out.println("*************************************************");

        //遍历map集合
        //通过迭代器遍历键值
        Iterator iterator = keySet.iterator();
        while (iterator.hasNext()) {
            Object key = iterator.next();
            //根据key获取value
            Object val = map4.get(key);
            System.out.println("Key:" + key + ";Value:" + val);
        }

        System.out.println("=================================================");

        //通过迭代器遍历键值对
        Iterator iterator2 = entrySet.iterator();
        while (iterator2.hasNext()) {
            Map.Entry entry = (Map.Entry) iterator2.next();
            System.out.println(entry.getKey() + "---" + entry.getValue());
        }

        System.out.println("=================================================");

        //通过foreach遍历
        for (Object entry : entrySet) {
            Map.Entry e = (Map.Entry) entry;
            System.out.println(e.getKey() + "---" + e.getValue());
        }
    }
}

HashMap

LinkedHashMap 在 HashMap基础之上，每个节点对象
static class Entry<K, V> extends HashMap.Node<K, V> {
  Entry<K, V> before, after;//before：指向上一个添加到集合中的元素 after：指向下一个添加到集合中的元素
  Entry(int hash, K key, V value, Node<K, V> next) {
   super(hash, key, value, next);
   }
}

package com.greedring;

import java.util.LinkedHashMap;

    }
}

TreeMap

可以对添加到集合中的元素进行排序(根据其键的自然顺序进行排序，或者根据创建映射时提供的 Comparator 进行排序)

package com.greedring;

import java.util.Comparator;
import java.util.Map;
import java.util.TreeMap;

public class Test {
    public static void main(String[] args) {
        Map map = new TreeMap(new Comparator() {
            @Override
            public int compare(Object o1, Object o2) {
                if (o1 instanceof User && o2 instanceof User) {
                    User u1 = (User) o1;
                    User u2 = (User) o2;
                    return u1.getName().compareTo(u2.getName());
                }
                return 0;
            }
        });

        map.put(new User("1", "Jack"), 1);
        map.put(new User("2", "Rose"), 6);
        map.put(new User("3", "Toz"), 3);
        map.put(new User("4", "Tony"), 4);
        map.put(new User("5", "Marry"), 7);

        System.out.println(map);
    }
}

面试题

简述Collection与Collections（集合的工具类）区别？
1.static boolean addAll(Collection c, T... elements) 将所有指定的元素添加到指定的集合中。 2.static void copy(List dest, List src) 将所有的元素从一个列表复制到另一个列表中。 3.static int frequency(Collection c, Object o)
返回指定集合中等于指定对象的元素的数目。
4.static <T extends Object & Comparable> T max(Collection coll) 返回最大元素的集合，根据其元素的自然排序。 5.static T max(Collection coll, Comparator comp) 返回给定集合的最大元素，根据指定的比较器诱导的顺序。 6.static > T min(Collection coll) 返回最小的元素的集合，根据其元素的自然排序。 7.static T min(Collection coll, Comparator comp) 返回给定集合的最小元素，根据指定的比较器诱导的顺序。 8.static void reverse(List list)
反转指定列表中元素的顺序。
9.static void shuffle(List list) 随机置换指定列表使用随机默认源。 10.static > void sort(List list) 指定列表为升序排序，根据其元素的 natural ordering。 11.static void sort(List list, Comparator c) 根据指定的比较器指定的顺序对指定的列表进行排序。 12.static void swap(List list, int i, int j)
在指定的列表中的指定位置上交换元素。

线程不安全集合转换为线程安全集合：
static Collection synchronizedCollection(Collection c)
返回由指定集合支持的同步（线程安全）集合。
static List synchronizedList(List list)
返回由指定列表支持的同步（线程安全）列表。
static <K,V> Map<K,V> synchronizedMap(Map<K,V> m)
返回由指定的Map支持的同步（线程安全）Map。
static Set synchronizedSet(Set s)
返回一个由指定集合支持的同步（线程安全）集。

package com.greedring;

import java.util.ArrayList;
import java.util.Arrays;
import java.util.Collections;
import java.util.Comparator;
import java.util.List;

public class Test {
    public static void main(String[] args) {
        List list = new ArrayList();
        // 1.向集合中添加若干个元素
        Collections.addAll(list, 1, true, "hello", "hello", false, 2, 3);
        System.out.println(list);

        List dest = Arrays.asList(new Object[7]);// java.lang.IndexOutOfBoundsException: Source does not fit in dest
        System.out.println(dest.size());
        // 2.拷贝集合元素
        Collections.copy(dest, list);
        System.out.println("dest=>" + dest);

        // 3.查找指定元素在集合中出现的次数
        int frequency = Collections.frequency(list, "hello");
        System.out.println("hello在list集合中出现的次数:" + frequency);

        // 4.查找集合最大值
        ArrayList l = new ArrayList();
        l.add(1);
        l.add(2);
        l.add(3);
        l.add(4);
        int i = (int) Collections.max(l, new Comparator() {

            @Override
            public int compare(Object o1, Object o2) {
                // TODO Auto-generated method stub
                if (o1 instanceof Integer && o2 instanceof Integer) {
                    Integer i1 = (Integer) o1;
                    Integer i2 = (Integer) o2;
                    if (i1 > i2) {
                        return -1;
                    } else if (i1 < i2) {
                        return 1;
                    } else {
                        return 0;
                    }
                }
                return 0;
            }
        });
        System.out.println("最大值:" + i);
    }
}

ArrayList

  • static void reverse(List<?> list)
    反转指定列表中元素的顺序。
  • static void shuffle(List<?> list)
    随机置换指定列表使用随机默认源。
  • static <T extends Comparable<? super T>> void sort(List<T> list)
    指定列表为升序排序，根据其元素的 natural ordering。
  • static <T> void sort(List<T> list, Comparator<? super T> c)
    根据指定的比较器指定的顺序对指定的列表进行排序。
  • static void swap(List<?> list, int i, int j)
    在指定的列表中的指定位置上交换元素。
package com.greedring;

import java.util.ArrayList;
import java.util.Collections;
import java.util.Date;
import java.util.List;

public class Test {
    public static void main(String[] args) {
        List list = new ArrayList();
        list.add(1);
        list.add(true);
        list.add(false);
        list.add(new Date());

        //反转
        Collections.reverse(list);
        System.out.println(list);

        //打乱集合中元素
        Collections.shuffle(list);
        System.out.println(list);
        Collections.shuffle(list);
        System.out.println(list);
        Collections.shuffle(list);
        System.out.println(list);

        System.out.println("=================================================");

        List l2 = new ArrayList();// 默认实现了Compareable接口，默认升序
        /*
         * l2.add(12); l2.add(6); l2.add(8); l2.add(22); l2.add(34); l2.add(5);
         * l2.add(7); l2.add(11);
         */
        l2.add(new User("2", "Jack"));
        l2.add(new User("1", "Jack"));
        l2.add(new User("6", "Jack"));
        l2.add(new User("5", "Jack"));

        Collections.sort(l2);
        //Collections.sort(list, c);

        System.out.println(l2);

        System.out.println("=================================================");

        list = new ArrayList();
        list.add(1);
        list.add(true);
        list.add(false);
        list.add(new Date());

        System.out.println(list);
        //交换位置
        Collections.swap(list, 1, 2);
        System.out.println(list);
    }
}
