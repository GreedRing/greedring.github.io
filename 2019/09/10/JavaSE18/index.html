双列集合

Map

接口，双列集合（存储一对一对的数据）类似函数y=f(x)
|—- HashMap:Map的主要实现类，线程不安全，效率高，可以存储为null的key和value
|—- LinkedHashMap：HashMap的子类，能够按照元素的添加顺序进行遍历（因为每个元素包含了两个引用Entry<K,V> before, after;分别指向上一个元素以及下一个元素）
如果涉及到频繁遍历操作时，LinkedHashMap性能高于HashMap
|—- TreeMap:可以对添加到集合中的元素进行排序（按照key的自然排序或者如果key是对象的话，按照对象的指定属性进行排序【自然排序、定制排序】）
|—- Hashtable:古老的双列集合，线程安全的（同步方法），效率低 不可以存储为null的key和value
|—- Properties:操作属性配置文件的，key、value都必须是String类型（IO）

Map、key、value的特点
以HashSet为例
Key是无序且不可重复，如果需要获取一个Map的所有的Key，可以使用的是Set结构
Value是无序可重复的，如果需要获取一个Map的所有Value，可以使用Collection结构存储
一个Map中的Entry对象无序且不可重复的，如果需要获取一个Map的所有键值对(Entry)，可以使用Set结构

Hash底层实现原理
HashMap map = new HashMap();
当创建HashMap时，底层默认创建一个Entry数组，并进行默认初始化，初始化长度为16

//之前已经存储了部分key…value
map.put(k1,v1)
先调用k1对象的hashCode方式，，根据k1的哈希值通过某种算法，得到元素需要再底层数组存储的位置（索引0-）
判断当前索引位置上是否有元素，如果当前索引位置上没有元素，k1，v1直接添加成功[情况1]
判断当前索引位置上存在一个或多个元素，将k1的哈希值依次与这一个或多个已经存在的key的哈希值进行比较，如果k1的哈希值与已存在元素的哈希值都不同，k1、v1添加成功[情况2]
如果k1的哈希值与已经存在的元素中的任意一个元素的key的哈希值相同(k2,v2)
调用equals方法比较k1、k2，如果返回false，k1、v1添加成功[情况3]
如果equals方法返回true，那么v1值将覆盖v2[情况4]

1.对于情况2、3多个元素再同一个数组中的位置使用链表进行存储
2.对于put过程（添加元素），可能会涉及到扩容操作，当数组使用长度超过临界值（而并非是数组长度），并且当前数组位置不为空的时候，需要进行扩容，扩容到原来的2倍，将将原有集合中数据拷贝到新的集合中

术语（常量）
DEFAULT_INITIAL_CAPACITY，默认初始化长度 16
DEFAULT_LOAD_FACTOR 默认加载因子 0.75（计算临界值（threshold）=16*0.75=12）
TREEIFY_THRESHOLD JDK1.8 桶中节点数量树化最大值（链表->红黑树）
MIN_TREEIFY_CAPACITY：64 树化数组的最小长度

JDK1.8 与 JDK 1.7区别：
（1）JDK1.7中底层是Entry[]数组 JDK1.8中底层是Node<K,V>[] table
static class Node<K,V> implements Map.Entry<K,V> {
final int hash;
final K key;
V value;
Node<K,V> next;
(2)JDK1.7中当调用HashMap()构造函数时，底层会创建一个长度为16的Entry数组，而JDK1.8中当调用构造函数时，并不会初始化底层数组
当第一次调用put操作时，初始化长度为16的Node[] 数组
（3）JDK1.7中桶中的数据是新增数据指向原有数据，JDK1.8是原有数据指向新增数据
（4）JDK1.8在某种情况（当前数组索引下节点的长度超过了8并且数组长度超过了64）下，会将桶中（数组中某个索引位置）的数据由链表结构转换为树形结构（树化）

Map方法

1.map的操作（新增、删除、修改）
void clear()
从这个映射中移除所有的映射（可选操作）。
V put(K key, V value)
将指定的值与此映射中的指定键关联（可选操作）。
void putAll(Map<? extends K,? extends V> m)
从指定的映射到这个Map（可选操作）复制所有的映射。
V remove(Object key)
如果存在（可选操作），则从该Map中移除一个键的映射。

2.map获取
boolean containsKey(Object key)
返回 true如果这Map包含一个指定的键映射。
boolean containsValue(Object value)
返回 true如果映射到指定的值的一个或多个键。
V get(Object key)
返回指定的键映射的值，或 null如果这个Map不包含的键映射。
int size()
返回这个映射中的键值映射的数目。
boolean isEmpty()
返回 true如果这个Map不包含键值的映射。
boolean equals(Object o)
将指定的对象与此映射的相等性进行比较。

3.map的元操作
Set<Map.Entry<K,V>> entrySet()
返回一个 Set视图的映射包含在这个Map。
Set keySet()
返回一个 Set的关键视图包含在这个Map。
Collection values()
返回一个 Collection视图的值包含在这个Map。

        //1.3 清除
        //map.clear();//移除集合中内容 （注意移除后并不是null）
        //System.out.println(map);

        //1.4 以某一个key移除键值对,并返回移除的value
        //Object remove = map.remove(1);
        //System.out.println("remove:"+remove);
        //System.out.println(map);

        //2.获取
        //2.1 根据指定key获取value
        Object object = map.get(1);
        System.out.println(object);
        Object object2 = map.get(new User("1", "jack"));
        System.out.println(object2);
        Object object3 = map.get("666");//如果不存在某个key返回null
        System.out.println(object3);
        System.out.println("--------------");
        //2.2 boolean containsKey(Object key) 判断key是否存在 （key使用set存储） 底层调用hashCode、equals
        boolean containsKey = map.containsKey("bb");
        System.out.println(containsKey);
        boolean containsKey2 = map.containsKey(new User("1", "jack"));
        System.out.println(containsKey2);
        System.out.println("--------------");
        //2.3 boolean containsValue(Object value)  判断value是否存在  （value使用collection存储） 底层调用equals
        boolean containsValue = map.containsValue("CC");
        System.out.println(containsValue);
        boolean containsValue2 = map.containsValue(new User("1", "Jack"));
        System.out.println(containsValue2);
        System.out.println("===================");
        //2.4 获取键值对的数量
        System.out.println(map.size());
        //2.5 判断集合中是否包含键值对
        map.clear();
        System.out.println(map.isEmpty());
        System.out.println("----------------");
        //2.6 判断两个集合中元素是否相同
        Map m1 = new HashMap();
        m1.put("a", "AA");
        m1.put("b", "BB");
        m1.put("c", "CC");
        Map m2 = new HashMap();
        m2.put("a", "AA");
        m2.put("b", "BB");
        m2.put("c", "CC");
        m2.put("d", "CC");
        System.out.println(m1.equals(m2));

        System.out.println("=======================");
        //1.map元操作
        //1.1 获取所有的key
        Set keySet = m2.keySet();
        System.out.println(keySet);

        //1.2获取所有的value
        Collection values = m2.values();
        System.out.println(values);

        //1.3获取所有的键值对
        Set entrySet = m2.entrySet();
        System.out.println(entrySet);

        System.out.println("===================");
        //遍历map集合
        //方式一(1/2)：
        Iterator iterator = keySet.iterator();
        while (iterator.hasNext()) {
            Object key = iterator.next();
            //根据key获取value
            Object val = m2.get(key);
            System.out.println("Key:" + key + ";Value:" + val);
        }

        System.out.println("=========");
        //方式二(1)：
        Iterator iterator2 = entrySet.iterator();
        while (iterator2.hasNext()) {
            Map.Entry entry = (Map.Entry) iterator2.next();
            System.out.println(entry.getKey() + "---" + entry.getValue());
        }

        System.out.println("========");

        //方式二(2)：
        for (Object entry : entrySet) {
            Map.Entry e = (Map.Entry) entry;
            System.out.println(e.getKey() + "---" + e.getValue());
        }
    }
}

HashMap

Map的主要实现类，线程不安全，效率高，可存储为null的key和value
LinkedMapHashMap的子类，能够安装元素的添加顺序进行遍历（因为每个元素包含了两个引用）
如果涉及到频繁遍历操作时，LinkedHashMap性能高于HashMap

LinkedHashMap 在 HashMap基础之上，每个节点对象
static class Entry<K,V> extends HashMap.Node<K,V> {
Entry<K,V> before, after;//before:指向上一个添加到集合中的元素 after:指向下一个添加到集合中的元素
Entry(int hash, K key, V value, Node<K,V> next) {
super(hash, key, value, next);
}
}

TreeMap

可以对添加到集合中的元素进行排序（按照key的自然排序或者如果key是对象的话，

package com.greedring;

import java.util.Comparator;
import java.util.Map;
import java.util.TreeMap;

public class Test {
    public static void main(String[] args) {
        Map map = new TreeMap(new Comparator() {
            @Override
            public int compare(Object o1, Object o2) {
                if (o1 instanceof User && o2 instanceof User) {
                    User u1 = (User) o1;
                    User u2 = (User) o2;
                    return u1.getName().compareTo(u2.getName());
                }
                return 0;
            }
        });

        map.put(new User("1", "Jack"), 1);
        map.put(new User("2", "Rose"), 6);
        map.put(new User("3", "Toz"), 3);
        map.put(new User("4", "Tony"), 4);
        map.put(new User("5", "Marry"), 7);

        System.out.println(map);
    }
}

1.static boolean addAll(Collection c, T... elements) 将所有指定的元素添加到指定的集合中。 2.static void copy(List dest, List src) 将所有的元素从一个列表复制到另一个列表中。 3.static int frequency(Collection c, Object o)
返回指定集合中等于指定对象的元素的数目。
4.static <T extends Object & Comparable> T max(Collection coll) 返回最大元素的集合，根据其元素的自然排序。 5.static T max(Collection coll, Comparator comp) 返回给定集合的最大元素，根据指定的比较器诱导的顺序。 6.static > T min(Collection coll) 返回最小的元素的集合，根据其元素的自然排序。 7.static T min(Collection coll, Comparator comp) 返回给定集合的最小元素，根据指定的比较器诱导的顺序。 8.static void reverse(List list)
反转指定列表中元素的顺序。
9.static void shuffle(List list) 随机置换指定列表使用随机默认源。 10.static > void sort(List list) 指定列表为升序排序，根据其元素的 natural ordering。 11.static void sort(List list, Comparator c) 根据指定的比较器指定的顺序对指定的列表进行排序。 12.static void swap(List list, int i, int j)
在指定的列表中的指定位置上交换元素。

package com.greedring;

import java.util.ArrayList;
import java.util.Arrays;
import java.util.Collections;
import java.util.Comparator;
import java.util.List;

public class Test {
    public static void main(String[] args) {
        List list = new ArrayList();
        // 1.向集合中添加若干个元素
        Collections.addAll(list, 1, true, "hello", "hello", false, 2, 3);
        System.out.println(list);

        List dest = Arrays.asList(new Object[7]);// java.lang.IndexOutOfBoundsException: Source does not fit in dest
        System.out.println(dest.size());
        // 2.拷贝集合元素
        Collections.copy(dest, list);
        System.out.println("dest=>" + dest);

        // 3.查找指定元素在集合中出现的次数
        int frequency = Collections.frequency(list, "hello");
        System.out.println("hello在list集合中出现的次数:" + frequency);

        // 4.查找集合最大值
        ArrayList l = new ArrayList();
        l.add(1);
        l.add(2);
        l.add(3);
        l.add(4);
        int i = (int) Collections.max(l, new Comparator() {

            @Override
            public int compare(Object o1, Object o2) {
                // TODO Auto-generated method stub
                if (o1 instanceof Integer && o2 instanceof Integer) {
                    Integer i1 = (Integer) o1;
                    Integer i2 = (Integer) o2;
                    if (i1 > i2) {
                        return -1;
                    } else if (i1 < i2) {
                        return 1;
                    } else {
                        return 0;
                    }
                }
                return 0;
            }
        });
        System.out.println("最大值:" + i);
    }
}

package com.greedring;

import java.util.ArrayList;
import java.util.Collections;
import java.util.Date;
import java.util.List;

public class Test {
    public static void main(String[] args) {
        List list = new ArrayList();
        list.add(1);
        list.add(true);
        list.add(false);
        list.add(new Date());

        // 反转
        Collections.reverse(list);
        System.out.println(list);

        // 打乱集合中元素
        Collections.shuffle(list);
        System.out.println(list);
        Collections.shuffle(list);
        System.out.println(list);
        Collections.shuffle(list);
        System.out.println(list);

        System.out.println("=====================");

        List l2 = new ArrayList();// 默认实现了Compareable接口，默认升序
        /*
         * l2.add(12); l2.add(6); l2.add(8); l2.add(22); l2.add(34); l2.add(5);
         * l2.add(7); l2.add(11);
         */
        l2.add(new User("2", "Jack"));
        l2.add(new User("1", "Jack"));
        l2.add(new User("6", "Jack"));
        l2.add(new User("5", "Jack"));

        Collections.sort(l2);
        // Collections.sort(list, c);

        System.out.println(l2);

        System.out.println("==============");

        list = new ArrayList();
        list.add(1);
        list.add(true);
        list.add(false);
        list.add(new Date());

        System.out.println(list);
        // 交换位置
        Collections.swap(list, 1, 2);
        System.out.println(list);
    }
}
