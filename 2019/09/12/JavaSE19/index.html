泛型与通配符

2019年09月12日 1.1k 字 大概 5 分钟 Java JavaSE

泛型、通配符

泛型

  • 泛型可以灵活的将数据类型应用到不同的类、方法、接口中，并将数据类型作为参数传递(泛型类、泛型接口、泛型方法)
  • 泛型是数据类的一部分，我们将类名与泛型合并一起看作数据类型。
  • 泛型定义：定义泛型可以在类中预支地使用未知数据类型。
  • 泛型的使用：一般在创建对象时，将未知的类型确定为具体的类型。当没有指定泛型时，默认类型为Object类型。

泛型类

当创建该类的对象时，传入类型，此时类上的泛型被确定
class 类名<E, T, ……>{   //<>中的泛型可以写无数个
  //E,T……能够做为类型在该类内部被使用
}

泛型方法

在调用泛型方法并传入实参的时候才能确定方法上形参类型
修饰符<T, Q, E, ……>返回值类型方法名(T t, Q q, ……){
//方法上的泛型定义在返回值的前面
}

泛型接口

interface 接口名<Q, E, R, ……>{
public abstract void method(T t);//假设定义一个方法
}

  • 方式一：
    实现类在实现接口时，指定泛型的实际类型

    public class MyUserClass implements 接口名<User>{
   @Override
   public void printMessage(User obj) {
   }    
}

  • 方式二：
    实现类依旧不指定泛型的实际类型，实现类也需要是一个泛型类

    class 类名<T> implements 接口名<T>{
   //由于没有指定接口中具体的数据类型,所以方法中仍然使用这种不确定的数据类型
   @Override
   public void method(T t) {    
   }
}

泛型的优点：

  • 提高程序的安全性
  • 将运行期问题转移到了编译期
  • 省去了类型强转的麻烦

通配符

  • 协变 <? extends T>
    逆变 <? super T>
    不变 T

  • 协变只能get，不能add（可以add null）
    逆变能add，不能get（可以用Object型的数据去get）
    不变能随便get和add

  • 协变 List<? extends Object> object = new ArrayList();
    逆变 List<? super String> object = new ArrayList

package com.greedring;

import java.util.ArrayList;
import java.util.Collections;
import java.util.List;

public class Test {
    public static void main(String[] args) {

        List<Student> list = new ArrayList<>();
        list.add(new Student(2, 60));
        list.add(new Student(3, 88));

        List<Student.Pupil> pupil = new ArrayList<>();

        //协变  向上限定  Student本身或者Student的子类
        List<? extends Student> l1 = null;

        l1 = list;
        //l1 = pupil;

        //取值
        Student student = l1.get(0);

        System.out.println(student);

        //设值
        //l1.add(new Student(2,90)); //?代表某个确定的Student子类或本身，无法添加除null以外的元素

        List<Object> os = new ArrayList<>();
        //l1 = os; //编译报错，os中元素Object是l1中元素Student的父类

        System.out.println("==============================================");

        //逆变  向下限定  集合中元素必须是Student以及Student的父类
        List<? super Student> l2 = null;
        l2 = list;

        //l2 = pupil; //编译报错

        l2 = os; //编译通过

        //取值
        Object object = l2.get(0);
        System.out.println(object);//?代表某个确定的Student父类或自身，只能用Object去接

        //设值
        l2.add(new Student(6, 66));

        System.out.println(l2);

        System.out.println(list);

        List<Student> ls = new ArrayList<>();
        Collections.addAll(ls, new Student.Pupil());

    }

    public static void printArray(List<?> list) {
        System.out.println(list);
        //list.add(12);
    }
}
