Idea下载jar包

2019年09月24日 107 字 大概 1 分钟 Java jar包下载

使用maven下载jar包

Maven

Idea新建项目，选择Maven，依次填写相关属性。

在新建Maven项目的红标处添加与

进到MVNRepository官网(点我打开)，搜索找到需要下载的jar包。

将框中的代码粘贴到<dependencies>与</dependencies>中即可通过Idea自动下载所需jar包。

多个jar包在<dependencies>与</dependencies>中依次添加。
